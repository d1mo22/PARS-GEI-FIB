## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits de l'especialitat de Computació que explora algorismes, les seves propietats i eficiència. El curs es basa en coneixements fonamentals per introduir tècniques avançades de disseny algorítmic i estructures de dades.

### Temes Principals
- **Fonaments Algorítmics**: Notació asimptòtica, dividir i vèncer, algorismes de grafs
- **Mètodes Greedy**: Algorisme de Dijkstra, arbres generadors mínims (Kruskal, Prim)
- **Programació Dinàmica**: Floyd-Warshall, distància d'edició, problemes de la motxilla
- **Fluxos de Xarxa**: Algorisme de Ford-Fulkerson, teorema de flux màxim/tall mínim
- **Estructures de Dades Avançades**: Heaps de Fibonacci, filtres de Bloom, estructures mètriques

### Objectius d'Aprenentatge
- Aplicar algorismes greedy per resoldre problemes d'optimització
- Utilitzar programació dinàmica per a reptes computacionals complexos
- Comprendre algorismes de flux de xarxa i les seves aplicacions
- Aprofitar l'aleatorització en el disseny d'algorismes
- Implementar estructures de dades avançades per a problemes especialitzats

## About the Subject

This is a 6-credit mandatory course in the Computer Science specialization that explores algorithms, their properties and efficiency. The course builds on foundational knowledge to introduce advanced algorithmic design techniques and data structures.

### Main Topics
- **Algorithmic Foundations**: Asymptotic notation, divide-and-conquer, graph algorithms
- **Greedy Methods**: Dijkstra's algorithm, minimum spanning trees (Kruskal, Prim)
- **Dynamic Programming**: Floyd-Warshall, edit distance, knapsack problems
- **Network Flows**: Ford-Fulkerson algorithm, max-flow/min-cut theorem
- **Advanced Data Structures**: Fibonacci heaps, bloom filters, metric structures

### Learning Objectives
- Apply greedy algorithms to solve optimization problems
- Use dynamic programming for complex computational challenges
- Understand network flow algorithms and their applications
- Leverage randomization in algorithm design
- Implement advanced data structures for specialized problems
