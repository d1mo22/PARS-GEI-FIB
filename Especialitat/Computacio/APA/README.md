## Sobre l'Assignatura

Aquesta assignatura se centra en desenvolupar teories, tècniques i algorismes que permeten als sistemes modificar el seu comportament mitjançant inferència inductiva a partir de dades incompletes. Connecta estadística, intel·ligència artificial, programació i optimització.

### Temes Principals

**Aprenentatge Supervisat:**
- Mètodes lineals (regressió, LDA, regressió logística)
- Enfocaments no lineals (xarxes neuronals, SVMs, arbres de decisió)
- Tècniques de reducció de dimensionalitat

**Aprenentatge No Supervisat:**
- Algorismes de clustering (k-means, EM)

**Aprenentatge per Reforç:**
- Processos de Markov i mètodes de diferència temporal

**Fonaments Teòrics:**
- Compromís biaix-variància, sobreajustament/subajustament, selecció de models

### Objectius d'Aprenentatge
- Formular problemes d'aprenentatge automàtic a partir de dades
- Organitzar fluxos de treball de solucions i defensar decisions tècniques
- Interpretar resultats i aplicar tècniques clau
- Implementar algorismes utilitzant Python

## About the Subject

This subject focuses on developing theories, techniques, and algorithms enabling systems to modify their behavior through inductive inference from incomplete data. It bridges statistics, artificial intelligence, programming, and optimization.

### Main Topics

**Supervised Learning:**
- Linear methods (regression, LDA, logistic regression)
- Nonlinear approaches (neural networks, SVMs, decision trees)
- Dimensionality reduction techniques

**Unsupervised Learning:**
- Clustering algorithms (k-means, EM)

**Reinforcement Learning:**
- Markov processes and temporal difference methods

**Theoretical Foundations:**
- Bias-variance tradeoff, overfitting/underfitting, model selection

### Learning Objectives
- Formulate machine learning problems from data
- Organize solution workflows and defend technical choices
- Interpret results and apply key techniques
- Implement algorithms using Python
