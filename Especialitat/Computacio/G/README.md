## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits centrada en visualització interactiva 3D i renderització realista d'escenes. Els estudiants desenvolupen aplicacions gràfiques utilitzant C++, OpenGL i GLSL.

### Temes Principals
- **Fonaments de visualització**: Emissió de llum, reflexió, transmissió; comportament difús i especular
- **Pipeline gràfic**: Transformacions geomètriques, processament de vèrtexs, rasterització, operacions de fragment
- **Renderització avançada**: Desenvolupament de shaders, texturització 2D, shadow mapping, environment mapping
- **Il·luminació global**: Fonaments de radiometria, BSDF, equacions de renderització, variants de ray-tracing
- **Interacció 3D**: Selecció d'objectes, manipulació, navegació de càmera
- **Estructures de dades espacials**: Optimització d'eficiència per a ray-tracing

### Objectius d'Aprenentatge
- Comprendre el procés complet de visualització per projecció
- Dominar models d'il·luminació local i global
- Desenvolupar programes de shaders en GLSL
- Implementar tècniques d'interacció amb escenes 3D
- Crear algorismes per a efectes realistes: textures, ombres, reflexions, transparència
- Implementar ray-tracing i mètodes d'intersecció raig-geometria

## About the Subject

This is a 6-credit mandatory course focusing on interactive 3D visualization and realistic scene rendering. Students develop graphics applications using C++, OpenGL, and GLSL.

### Main Topics
- **Visualization fundamentals**: Light emission, reflection, transmission; diffuse and specular behavior
- **Graphics pipeline**: Geometric transformations, vertex processing, rasterization, fragment operations
- **Advanced rendering**: Shader development, 2D texturing, shadow mapping, environment mapping
- **Global illumination**: Radiometry basics, BSDF, rendering equations, ray-tracing variants
- **3D interaction**: Object selection, manipulation, camera navigation
- **Spatial data structures**: Efficiency optimization for ray-tracing

### Learning Objectives
- Understand the complete projection visualization process
- Master local and global illumination models
- Develop shader programs in GLSL
- Implement 3D scene interaction techniques
- Create algorithms for realistic effects: textures, shadows, reflections, transparency
- Implement ray-tracing and ray-geometry intersection methods
