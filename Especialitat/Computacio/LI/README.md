## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits de l'especialitat de Computació que proporciona una base sòlida de lògica per a informàtics. El curs explora com la lògica s'aplica en bases de dades, complexitat computacional, llenguatges de programació, IA i verificació de sistemes.

### Temes Principals
- **Lògica Proposicional**: Fórmules, taules de veritat, satisfactibilitat, formes normals
- **Mètodes de Deducció**: Algorisme DPLL, resolució, SAT solvers
- **Lògica de Primer Ordre**: Quantificadors, unificació, Skolemització
- **Programació Lògica**: Prolog, resolució SLD, backtracking
- **Programació amb Restriccions**: CLP(X), assignació de recursos, problemes de planificació

### Objectius d'Aprenentatge
- Comprendre fonaments de lògica: sintaxi (fórmules) i semàntica (interpretacions)
- Dominar lògiques proposicional i de primer ordre amb aplicacions pràctiques
- Aplicar mètodes de deducció com Davis-Putnam i resolució
- Utilitzar SAT solvers i programació lògica (Prolog) per a problemes reals
- Expressar problemes NP-complets com reptes de satisfactibilitat
- Implementar programació lògica amb restriccions per a optimització

## About the Subject

This is a 6-credit mandatory course in the Computer Science specialization that provides a solid foundation of logic for computer scientists. The course explores how logic applies across databases, computational complexity, programming languages, AI, and system verification.

### Main Topics
- **Propositional Logic**: Formulas, truth tables, satisfiability, normal forms
- **Deduction Methods**: DPLL algorithm, resolution, SAT solvers
- **First-Order Logic**: Quantifiers, unification, Skolemization
- **Logical Programming**: Prolog, SLD resolution, backtracking
- **Constraint Programming**: CLP(X), resource allocation, scheduling problems

### Learning Objectives
- Understand logic fundamentals: syntax (formulas) and semantics (interpretations)
- Master propositional and first-order logics with practical applications
- Apply deduction methods like Davis-Putnam and resolution
- Use SAT solvers and logical programming (Prolog) for real problems
- Express NP-complete problems as satisfiability challenges
- Implement constraint logic programming for optimization
