# LP - Llenguatges de Programacio (Programming Languages) - Q6 Track

## About the Subject

This is the Q6 iteration of the Programming Languages course (6 credits), exploring programming language fundamentals, compilation processes, and multiple programming paradigms.

### Main Topics
- **Compilation Fundamentals**: Lexical/syntactic/semantic analysis, compilers vs. interpreters
- **Functional Programming**: Pattern matching, eager/lazy evaluation, Haskell
- **Type Systems**: Type safety across different programming paradigms
- **Higher-Order Programming**: Functions as first-class values
- **Scripting Languages**: Python and practical applications

### Learning Objectives
- Understand the different phases of compilation
- Distinguish programming language characteristics
- Master functional programming concepts
- Learn scripting languages independently
- Apply formal specification methods
