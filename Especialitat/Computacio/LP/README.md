## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que explora els fonaments dels llenguatges de programació, processos de compilació i paradigmes de programació funcional.

### Temes Principals
- **Fonaments de Compilació**: Anàlisi lèxica/sintàctica/semàntica, compiladors vs. intèrprets
- **Programació Funcional**: Pattern matching, avaluació eager/lazy, introducció a Haskell
- **Sistemes de Tipus**: Seguretat de tipus en diferents paradigmes de programació
- **Programació d'Ordre Superior**: Funcions com a valors de primera classe
- **Especificació i Modelització**: Ús de llenguatges funcionals per al disseny de sistemes
- **Llenguatges de Scripting**: Python i aplicacions pràctiques

### Objectius d'Aprenentatge
- Comprendre les diferents fases de compilació (anàlisi lèxica, sintàctica, semàntica)
- Distingir característiques dels llenguatges de programació i justificar la selecció de llenguatges
- Dominar conceptes de programació funcional i Haskell
- Aprendre llenguatges de scripting de forma independent
- Aplicar mètodes d'especificació formal utilitzant enfocaments funcionals

## About the Subject

This is a 6-credit mandatory course that explores programming language fundamentals, compilation processes, and functional programming paradigms.

### Main Topics
- **Compilation Fundamentals**: Lexical/syntactic/semantic analysis, compilers vs. interpreters
- **Functional Programming**: Pattern matching, eager/lazy evaluation, Haskell introduction
- **Type Systems**: Type safety across different programming paradigms
- **Higher-Order Programming**: Functions as first-class values
- **Specification & Modeling**: Using functional languages for system design
- **Scripting Languages**: Python and practical applications

### Learning Objectives
- Understand the different phases of compilation (lexical, syntactic, semantic analysis)
- Distinguish programming language characteristics and justify language selection
- Master functional programming concepts and Haskell
- Learn scripting languages independently
- Apply formal specification methods using functional approaches
