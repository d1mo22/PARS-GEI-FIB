# AS - Arquitectura del Software (Software Architecture)

## About the Subject

This is a 6-credit mandatory specialization course in Software Engineering that capacitates students to design and implement software systems through application of design techniques, principles, and patterns.

### Main Topics
- **Introduction**: Development methodologies, architectural concepts, design patterns
- **Architectural Patterns**: Layered architecture, object-oriented design principles
- **System Design**: Domain layer design, presentation layer patterns, data persistence strategies, service integration

### Learning Objectives
- Distinguish logical and physical system architectures
- Apply layered and object-oriented architectural patterns
- Design system components (domain, presentation, data layers)
- Use UML for design representation
- Evaluate and critique software designs critically
- Integrate existing software services into designs
