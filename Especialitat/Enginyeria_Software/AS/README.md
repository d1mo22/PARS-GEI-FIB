## Sobre l'Assignatura

Assignatura obligatòria d'especialitat de 6 crèdits d'Enginyeria del Software que capacita els estudiants per dissenyar i implementar sistemes de software mitjançant l'aplicació de tècniques, principis i patrons de disseny.

### Temes Principals
- **Introducció**: Metodologies de desenvolupament, conceptes arquitectònics, patrons de disseny
- **Patrons Arquitectònics**: Arquitectura en capes, principis de disseny orientat a objectes
- **Disseny de Sistemes**: Disseny de capa de domini, patrons de capa de presentació, estratègies de persistència de dades, integració de serveis

### Objectius d'Aprenentatge
- Distingir arquitectures lògiques i físiques de sistemes
- Aplicar patrons arquitectònics en capes i orientats a objectes
- Dissenyar components del sistema (capes de domini, presentació, dades)
- Utilitzar UML per a representació de dissenys
- Avaluar i criticar dissenys de software de forma crítica
- Integrar serveis de software existents en els dissenys

## Sobre la Asignatura

Asignatura obligatoria de especialidad de 6 créditos de Ingeniería del Software que capacita a los estudiantes para diseñar e implementar sistemas de software mediante la aplicación de técnicas, principios y patrones de diseño.

### Temas Principales
- **Introducción**: Metodologías de desarrollo, conceptos arquitectónicos, patrones de diseño
- **Patrones Arquitectónicos**: Arquitectura en capas, principios de diseño orientado a objetos
- **Diseño de Sistemas**: Diseño de capa de dominio, patrones de capa de presentación, estrategias de persistencia de datos, integración de servicios

### Objetivos de Aprendizaje
- Distinguir arquitecturas lógicas y físicas de sistemas
- Aplicar patrones arquitectónicos en capas y orientados a objetos
- Diseñar componentes del sistema (capas de dominio, presentación, datos)
- Utilizar UML para representación de diseños
- Evaluar y criticar diseños de software de forma crítica
- Integrar servicios de software existentes en los diseños

## About the Subject

This is a 6-credit mandatory specialization course in Software Engineering that capacitates students to design and implement software systems through application of design techniques, principles, and patterns.

### Main Topics
- **Introduction**: Development methodologies, architectural concepts, design patterns
- **Architectural Patterns**: Layered architecture, object-oriented design principles
- **System Design**: Domain layer design, presentation layer patterns, data persistence strategies, service integration

### Learning Objectives
- Distinguish logical and physical system architectures
- Apply layered and object-oriented architectural patterns
- Design system components (domain, presentation, data layers)
- Use UML for design representation
- Evaluate and critique software designs critically
- Integrate existing software services into designs
