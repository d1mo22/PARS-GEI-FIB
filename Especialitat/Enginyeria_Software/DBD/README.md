# DBD - Disseny de Bases de Dades (Database Design)

## About the Subject

This 6-credit mandatory course trains students in designing and managing databases within software engineering specialization contexts, covering both operational and decision-support systems.

### Main Topics
- **Conceptual foundations**: Lifecycle, operational vs. decision systems
- **OLAP & data warehousing**: Multidimensional schemas, star-join models
- **Logical design**: UML-to-relational mapping, handling nulls
- **Design correctness**: Normalization, validation algorithms
- **Physical design**: Constraints, materialized views, indexing structures
- **Query optimization**: Semantic, syntactic, and physical approaches
- **Access structures**: B-trees, hashing, bitmap indexes
- **Administration**: Transactions, isolation levels, recovery mechanisms
- **NoSQL basics**: Alternatives to relational models

### Learning Objectives
- Master multidimensional database design and querying using SQL-99
- Implement logical-to-physical design with integrity constraints
- Analyze query execution plans and cost estimation
- Perform database administration tasks
