## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que entrena els estudiants en el disseny i gestió de bases de dades dins del context de l'especialitat d'enginyeria del software, cobrint tant sistemes operacionals com de suport a decisions.

### Temes Principals
- **Fonaments conceptuals**: Cicle de vida, sistemes operacionals vs. de decisió
- **OLAP i data warehousing**: Esquemes multidimensionals, models star-join
- **Disseny lògic**: Mapeig UML-a-relacional, gestió de nuls
- **Correcció del disseny**: Normalització, algorismes de validació
- **Disseny físic**: Restriccions, vistes materialitzades, estructures d'indexació
- **Optimització de consultes**: Enfocaments semàntics, sintàctics i físics
- **Estructures d'accés**: B-trees, hashing, índexs bitmap
- **Administració**: Transaccions, nivells d'aïllament, mecanismes de recuperació
- **Fonaments NoSQL**: Alternatives als models relacionals

### Objectius d'Aprenentatge
- Dominar el disseny de bases de dades multidimensionals i consultes amb SQL-99
- Implementar disseny lògic-a-físic amb restriccions d'integritat
- Analitzar plans d'execució de consultes i estimació de costos
- Realitzar tasques d'administració de bases de dades

## About the Subject

This 6-credit mandatory course trains students in designing and managing databases within software engineering specialization contexts, covering both operational and decision-support systems.

### Main Topics
- **Conceptual foundations**: Lifecycle, operational vs. decision systems
- **OLAP & data warehousing**: Multidimensional schemas, star-join models
- **Logical design**: UML-to-relational mapping, handling nulls
- **Design correctness**: Normalization, validation algorithms
- **Physical design**: Constraints, materialized views, indexing structures
- **Query optimization**: Semantic, syntactic, and physical approaches
- **Access structures**: B-trees, hashing, bitmap indexes
- **Administration**: Transactions, isolation levels, recovery mechanisms
- **NoSQL basics**: Alternatives to relational models

### Learning Objectives
- Master multidimensional database design and querying using SQL-99
- Implement logical-to-physical design with integrity constraints
- Analyze query execution plans and cost estimation
- Perform database administration tasks
