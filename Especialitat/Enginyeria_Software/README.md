# Enginyeria del Software (Software Engineering Specialization)

This folder contains subjects from the Software Engineering specialization at FIB.

## Subjects

| Code | Subject | Description |
|------|---------|-------------|
| AS | [Arquitectura del Software](./AS/) | Software architecture, design patterns, layered systems |
| DBD | [Disseny de Bases de Dades](./DBD/) | Database design, OLAP, query optimization, NoSQL |
| ER | [Enginyeria de Requisits](./ER/) | Requirements engineering, use cases, stakeholder analysis |

## Specialization Focus

The Software Engineering specialization emphasizes:
- Software design methodologies
- Architectural patterns and principles
- Requirements analysis and specification
- Database design and optimization
- Software quality and testing
