## Sobre l'Assignatura

Assignatura obligatòria de 7.5 crèdits que cobreix dues àrees principals: lògica matemàtica i matemàtiques discretes, amb èmfasi en teoria de nombres.

### Temes Principals
- **Lògica i Conceptes Fonamentals**: Proposicions, lògica de predicats, teoria de conjunts, relacions, classes d'equivalència, funcions
- **Teoria de Nombres**: Divisibilitat, nombres primers, algorisme d'Euclides, MCD/MCM, aritmètica modular, criptografia RSA

### Objectius d'Aprenentatge
- Comprendre el llenguatge matemàtic precís per a la comunicació científica
- Dominar les demostracions matemàtiques i els seus diferents tipus
- Aplicar el principi d'inducció per demostrar propietats
- Entendre la divisibilitat d'enters i l'algorisme d'Euclides
- Aplicar congruències en aplicacions computacionals

## About the Subject

This is a 7.5-credit mandatory course that covers two main areas: mathematical logic and discrete mathematics, with a focus on number theory.

### Main Topics
- **Logic & Foundational Concepts**: Propositions, predicate logic, set theory, relations, equivalence classes, functions
- **Number Theory**: Divisibility, prime numbers, Euclidean algorithm, GCD/LCM, modular arithmetic, RSA cryptography

### Learning Objectives
- Understand precise mathematical language for scientific communication
- Master mathematical proofs and their various types
- Apply the induction principle for proving properties
- Understand integer divisibility and the Euclidean algorithm
- Apply congruences in computational applications
