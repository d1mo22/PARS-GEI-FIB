## Sobre l'Assignatura

Assignatura fonamental de 7.5 crèdits que ensenya els fonaments de la programació imperativa utilitzant C++. Els estudiants aprenen conceptes bàsics i tècniques per construir programes en llenguatges imperatius.

### Temes Principals
- **Fonaments**: Variables, expressions, tipus de dades, instruccions C++
- **Flux de Control**: Iteració amb bucles for/while, condicionals
- **Programació Modular**: Funcions, procediments, àmbit, paràmetres
- **Algorismes**: Recorregut, cerca, recursivitat, ordenació (inserció, selecció, mergesort, quicksort), cerca binària
- **Estructures de Dades**: Vectors unidimensionals i multidimensionals

### Objectius d'Aprenentatge
- Dominar les construccions bàsiques de programació
- Aplicar metodologia de disseny top-down
- Implementar algorismes fonamentals incloent cerca binària i ordenació
- Treballar amb vectors i estructures de dades bàsiques

## About the Subject

This is a 7.5-credit foundational programming course teaching imperative language fundamentals using C++. Students learn basic concepts and techniques for building programs in imperative languages.

### Main Topics
- **Basics**: Variables, expressions, data types, C++ instructions
- **Control Flow**: Iteration with for/while loops, conditionals
- **Modular Programming**: Functions, procedures, scope, parameters
- **Algorithms**: Traversal, searching, recursion, sorting (insertion, selection, mergesort, quicksort), binary search
- **Data Structures**: One and multidimensional arrays

### Learning Objectives
- Master core programming constructs
- Apply top-down design methodology
- Implement fundamental algorithms including binary search and sorting
- Work with arrays and basic data structures
