# EC - Estructura de Computadors

<summary>Català</summary>

## Sobre l'Assignatura

Assignatura obligatòria de 7.5 crèdits que explora l'organització bàsica dels computadors que suporten llenguatges de programació d'alt nivell, centrant-se en tipus de dades i sentències de control.

### Temes Principals
- **Introducció**: Jerarquia de computadors, mètriques de rendiment
- **Assemblador i Tipus de Dades**: Fonaments de MIPS, representació de nombres
- **Traducció de Programes**: Operacions lògiques, flux de control, subrutines
- **Vectors i Matrius**: Emmagatzematge i accés seqüencial
- **Aritmètica**: Operacions amb enters i punt flotant IEEE754
- **Memòria Cau**: Mapatge directe, associativitat, caus multinivell
- **Memòria Virtual**: Traducció d'adreces, gestió de TLB
- **Excepcions/Interrupcions**: Suport hardware i gestió

### Objectius d'Aprenentatge
- Comprendre l'organització jeràrquica dels computadors
- Dominar els conjunts d'instruccions de processadors RISC
- Gestionar representacions aritmètiques d'enters i punt flotant
- Traduir programes d'alt nivell a llenguatge assemblador
- Entendre els sistemes de memòria cau i memòria virtual

## About the Subject

This is a 7.5-credit mandatory course exploring the basic organization of computers supporting high-level programming languages, focusing on data types and control statements.

### Main Topics
- **Introduction**: Computer hierarchy, performance metrics
- **Assembly & Data Types**: MIPS fundamentals, number representation
- **Program Translation**: Logic operations, control flow, subroutines
- **Arrays & Matrices**: Storage and sequential access
- **Arithmetic**: Integer and IEEE754 floating-point operations
- **Cache Memory**: Direct mapping, associativity, multi-level caches
- **Virtual Memory**: Address translation, TLB management
- **Exceptions/Interrupts**: Hardware support and handling

### Learning Objectives
- Understand hierarchical computer organization
- Master RISC processor instruction sets
- Handle integer and floating-point arithmetic representations
- Translate high-level programs to assembly language
- Understand cache and virtual memory systems
