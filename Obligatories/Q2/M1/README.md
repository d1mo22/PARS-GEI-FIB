## Sobre l'Assignatura

Assignatura obligatòria de 7.5 crèdits que continua desenvolupant el raonament matemàtic mentre introdueix dos temes essencials per als enginyers informàtics: teoria de grafs i àlgebra lineal.

### Temes Principals
- **Teoria de Grafs**: Conceptes bàsics de grafs, matrius d'adjacència/incidència, camins, connectivitat, grafs eulerians/hamiltonians, arbres i arbres generadors
- **Àlgebra Lineal**: Matrius, sistemes d'equacions lineals, espais/subespais vectorials, aplicacions lineals, valors propis, vectors propis, diagonalització

### Objectius d'Aprenentatge
- Desenvolupar capacitat de raonament crític, lògic i matemàtic
- Resoldre problemes mitjançant pensament abstracte i resolució pràctica
- Dominar conceptes i algorismes de teoria de grafs
- Aplicar tècniques d'àlgebra lineal en contextos computacionals

## About the Subject

This is a 7.5-credit mandatory course that continues developing mathematical reasoning while introducing two essential topics for computer engineers: graph theory and linear algebra.

### Main Topics
- **Graph Theory**: Basic graph concepts, adjacency/incidence matrices, paths, connectivity, Eulerian/Hamiltonian graphs, trees and spanning trees
- **Linear Algebra**: Matrices, systems of linear equations, vector spaces/subspaces, linear applications, eigenvalues, eigenvectors, diagonalization

### Learning Objectives
- Develop capacity for critical, logical and mathematical reasoning
- Solve problems through abstract thinking and practical problem-solving
- Master graph theory concepts and algorithms
- Apply linear algebra techniques in computational contexts
