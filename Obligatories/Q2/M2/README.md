# M2 - Matematiques II (Mathematics II)

## About the Subject

This is a 7.5-credit mandatory course designed to equip computer science students with fundamental concepts and techniques of mathematical calculation, focusing on functions of one and multiple variables.

### Main Topics
- **Single-variable functions**: Real numbers, sequences, continuity, derivatives, Taylor formulas, integration
- **Multivariable functions**: Topology basics, partial/directional derivatives, gradient vectors, Hessian matrices, constrained optimization via Lagrange multipliers

### Learning Objectives
- Master real numbers, equations, and inequalities
- Understand numerical sequences and convergence
- Apply continuity and differentiability theorems
- Use Taylor polynomials and approximation methods
- Perform integration techniques
- Work with multivariable functions and optimization
