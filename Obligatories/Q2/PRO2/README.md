## Sobre l'Assignatura

Assignatura de 7.5 crèdits que introdueix el disseny modular i la programació orientada a objectes utilitzant C++. Els estudiants aprenen estructures de dades lineals (piles, cues, llistes) i estructures d'arbre, amb èmfasi en el disseny d'algorismes iteratius i recursius.

### Temes Principals
- Estructures de dades lineals (piles, cues, llistes, mapes, conjunts)
- Estructures de dades basades en arbres (arbres binaris)
- Invariants de bucle i correcció de programes iteratius
- Disseny recursiu i generalització de funcions
- Principis de disseny modular i orientat a objectes
- Implementacions basades en punters i memòria dinàmica

### Objectius d'Aprenentatge
- Dissenyar classes de dades independents amb clara separació entre especificació i implementació
- Implementar estructures de dades enllaçades utilitzant punters
- Analitzar la correcció i eficiència d'algorismes
- Dissenyar programes modulars en C++

## About the Subject

This 7.5-credit course introduces modular design and object-oriented programming using C++. Students learn linear data structures (stacks, queues, lists) and tree structures, with emphasis on iterative and recursive algorithm design.

### Main Topics
- Linear data structures (stacks, queues, lists, maps, sets)
- Tree-based data structures (binary trees)
- Loop invariants and iterative program correctness
- Recursive design and function generalization
- Modular and object-oriented design principles
- Pointer-based implementations and dynamic memory

### Learning Objectives
- Design independent data classes with clear separation between specification and implementation
- Implement linked data structures using pointers
- Analyze algorithm correctness and efficiency
- Design modular C++ programs
