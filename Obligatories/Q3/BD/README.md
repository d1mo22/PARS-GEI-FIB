## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que proporciona una cobertura completa dels sistemes de bases de dades. Els estudiants adquireixen comprensió dels conceptes, models, tipus d'usuaris i llenguatges de bases de dades, amb èmfasi en el model relacional i la implementació SQL.

### Temes Principals
- **Fonaments de Bases de Dades**: Conceptes, models, objectius dels SGBD
- **Model Relacional**: Estructures de dades, operacions, regles d'integritat
- **SQL i Àlgebra Relacional**: Llenguatges de consulta i manipulació de bases de dades
- **Components Lògics**: Esquemes, dominis, taules, vistes, procediments emmagatzemats, triggers
- **Disseny de Bases de Dades**: Traducció d'UML a model relacional
- **Transaccions i Concurrència**: Propietats ACID, nivells d'aïllament, bloquejos
- **Emmagatzematge Físic**: Estructures de dades i mètodes d'accés
- **Introducció a NOSQL**: Visió general de sistemes de bases de dades alternatius

### Objectius d'Aprenentatge
- Dominar els components del model relacional i les operacions SQL
- Dissenyar i implementar bases de dades relacionals a partir de models UML
- Comprendre transaccions, control de concurrència i emmagatzematge físic
- Avaluar alternatives NOSQL

## Sobre la Asignatura

- **Componentes Lógicos**: Esquemas, dominios, tablas, vistas, procedimientos almacenados, triggers
- **Diseño de Bases de Datos**: Traducción de UML a modelo relacional
- **Transacciones y Concurrencia**: Propiedades ACID, niveles de aislamiento, bloqueos
- **Almacenamiento Físico**: Estructuras de datos y métodos de acceso

### Objetivos de Aprendizaje
- Dominar los componentes del modelo relacional y las operaciones SQL
- Diseñar e implementar bases de datos relacionales a partir de modelos UML
- Comprender transacciones, control de concurrencia y almacenamiento físico
- Evaluar alternativas NOSQL

## About the Subject

This is a 6-credit mandatory course that provides comprehensive coverage of database systems. Students gain understanding of database concepts, models, user types, and languages, with emphasis on the relational model and SQL implementation.

### Main Topics
- **Database Fundamentals**: Concepts, models, DBMS objectives
- **Relational Model**: Data structures, operations, integrity rules
- **SQL & Relational Algebra**: Query languages and database manipulation
- **Logical Components**: Schemas, domains, tables, views, stored procedures, triggers
- **Database Design**: UML to relational model translation
- **Transactions & Concurrency**: ACID properties, isolation levels, locking
- **Physical Storage**: Data structures and access methods
- **NOSQL Introduction**: Alternative database systems overview

### Learning Objectives
- Master relational model components and SQL operations
- Design and implement relational databases from UML models
- Understand transactions, concurrency control, and physical storage
- Evaluate NOSQL alternatives
