## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que examina el sistema d'entrada/sortida del computador en profunditat, centrant-se en les interfícies hardware-software i la seva implementació pràctica.

### Temes Principals
- Arquitectura de microcomputadors i pipelines d'execució d'instruccions
- Estructures de ports d'entrada/sortida i mecanismes de control
- Gestió d'interrupcions i rutines de servei
- Funcionalitat de temporitzadors/comptadors per a generació de polsos i senyals
- Interfícies de conversió analògic-digital i digital-analògic
- Protocols de comunicació sèrie (UART, SPI, I2C, USB)
- Jerarquies de bus i operacions d'Accés Directe a Memòria (DMA)

### Objectius d'Aprenentatge
- Programar subsistemes de microcontroladors
- Gestionar restriccions temporals
- Resoldre compromisos hardware-software
- Interpretar documentació tècnica
- Connectar principis teòrics amb implementació de sistemes encastats

## About the Subject

This 6-credit mandatory course examines the computer's input/output system in depth, focusing on hardware-software interfaces and their practical implementation.

### Main Topics
- Microcomputer architecture and instruction execution pipelines
- Input/output port structures and control mechanisms
- Interrupt handling and service routines
- Timer/counter functionality for pulse and signal generation
- Analog-to-digital and digital-to-analog conversion interfaces
- Serial communication protocols (UART, SPI, I2C, USB)
- Bus hierarchies and Direct Memory Access (DMA) operations

### Learning Objectives
- Program microcontroller subsystems
- Manage temporal constraints
- Resolve hardware-software tradeoffs
- Interpret technical documentation
- Bridge theoretical principles with embedded systems implementation
