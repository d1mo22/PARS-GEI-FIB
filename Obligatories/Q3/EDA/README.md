## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que introdueix l'anàlisi d'eficiència d'algorismes i les estructures de dades fonamentals. Els estudiants aprenen a avaluar la complexitat computacional utilitzant notació asimptòtica, i després estudien algorismes clàssics i les seves implementacions.

### Temes Principals
- Anàlisi d'algorismes i notació asimptòtica
- Algorismes de dividir i vèncer (mergesort, quicksort, multiplicació de matrius)
- Implementacions de diccionaris (vectors, llistes enllaçades, taules de hash, arbres balancejats)
- Cues de prioritat i heaps
- Algorismes de grafs (DFS, BFS, Dijkstra, Prim)
- Backtracking i cerca exhaustiva
- Introducció a P, NP i NP-completesa

### Objectius d'Aprenentatge
- Analitzar el cost d'algorismes en els casos millor, mitjà i pitjor
- Dissenyar solucions utilitzant tècniques de dividir i vèncer
- Implementar estructures de diccionari (taules de hash, BSTs, arbres AVL)
- Construir algorismes de grafs
- Entendre els límits computacionals a través de la teoria de NP-completesa

## About the Subject

This 6-credit mandatory course introduces algorithm efficiency analysis and foundational data structures. Students learn to evaluate computational complexity using asymptotic notation, then study classic algorithms and implementations.

### Main Topics
- Algorithm analysis and asymptotic notation
- Divide-and-conquer algorithms (mergesort, quicksort, matrix multiplication)
- Dictionary implementations (arrays, linked lists, hash tables, balanced trees)
- Priority queues and heaps
- Graph algorithms (DFS, BFS, Dijkstra, Prim)
- Backtracking and exhaustive search
- Introduction to P, NP, and NP-completeness

### Learning Objectives
- Analyze algorithm cost across best, average, and worst cases
- Design solutions using divide-and-conquer techniques
- Implement dictionary structures (hash tables, BSTs, AVL trees)
- Build graph algorithms
- Understand computational limits through NP-completeness theory
