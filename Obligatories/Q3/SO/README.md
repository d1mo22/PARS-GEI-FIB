## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que examina els sistemes operatius des de tres perspectives: serveis oferts a usuaris/programes, elements de disseny intern del kernel, i com l'execució concurrent afecta el rendiment del sistema.

### Temes Principals
- **Introducció als Sistemes Operatius**: Funcions, serveis, gestió de recursos
- **Gestió de Processos**: Cicle de vida de processos, planificació, canvi de context, concurrència, senyals
- **Gestió de Memòria**: Memòria virtual, traducció d'adreces, mecanismes de protecció
- **Gestió d'Entrada/Sortida**: Tipus de dispositius, independència de dispositius, sistemes de fitxers, i-nodes
- **Execució de Fils**: Abstraccions de fils i estructures de gestió

### Objectius d'Aprenentatge
- Comprendre les funcions del SO i la gestió de recursos (abstracció, virtualització, seguretat, eficiència)
- Comprendre mecanismes interns del SO a través d'estructures de dades i algorismes
- Entendre les interaccions entre hardware, biblioteques del sistema i programes d'usuari
- Aplicar mètriques d'avaluació per valorar la qualitat i rendiment del sistema

## About the Subject

This 6-credit mandatory course examines operating systems from three perspectives: services offered to users/programs, internal kernel design elements, and how concurrent execution affects system performance.

### Main Topics
- **Introduction to Operating Systems**: Functions, services, resource management
- **Process Management**: Process lifecycle, scheduling, context switching, concurrency, signals
- **Memory Management**: Virtual memory, address translation, protection mechanisms
- **Input/Output Management**: Device types, device independence, file systems, i-nodes
- **Thread Execution**: Thread abstractions and management structures

### Learning Objectives
- Understand OS functions and resource management (abstraction, virtualization, security, efficiency)
- Comprehend internal OS mechanisms through data structures and algorithms
- Grasp interactions between hardware, system libraries, and user programs
- Apply evaluation metrics to assess system quality and performance
