## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que ensenya com funcionen els computadors moderns i els fonaments del disseny i avaluació de computadors, emfatitzant l'impacte de les decisions de disseny en el rendiment, consum d'energia, cost i fiabilitat.

### Temes Principals
- Jerarquia de memòria i tècniques d'optimització
- Sistemes d'emmagatzematge i fiabilitat de dades
- Arquitectura del conjunt d'instruccions (ISA) i paradigmes RISC-CISC
- Tècniques de paral·lelisme de processadors (pipelining, superescalar, VLIW, vectorització, multithreading, multiprocessadors)
- Interacció hardware/software per a optimització del rendiment

### Objectius d'Aprenentatge
- Traduir codi d'alt nivell a assemblador (IA32) i enllaçar amb C utilitzant Linux ABI
- Descriure components de la jerarquia de memòria i millores de rendiment
- Avaluar la fiabilitat del sistema utilitzant mecanismes de detecció/correcció d'errors
- Avaluar el rendiment del codi considerant memòria, emmagatzematge, ISA i disseny del processador
- Avaluar el consum d'energia i l'impacte energètic en aplicacions
- Aplicar tècniques d'optimització per al rendiment i eficiència del codi

## About the Subject

This is a 6-credit compulsory course teaching how modern computers function and the fundamentals of computer design and evaluation, emphasizing the impact of design decisions on performance, power consumption, energy use, cost and reliability.

### Main Topics
- Memory hierarchy and optimization techniques
- Storage systems and data reliability
- Instruction set architecture (ISA) and RISC-CISC paradigms
- Processor parallelism techniques (pipelining, superscalar, VLIW, vectorization, multithreading, multiprocessors)
- Hardware/software interaction for performance optimization

### Learning Objectives
- Translate high-level code to assembly (IA32) and link with C using Linux ABI
- Describe memory hierarchy components and performance improvements
- Evaluate system reliability using error detection/correction mechanisms
- Assess code performance considering memory, storage, ISA, and processor design
- Evaluate power consumption and energy impact on applications
- Apply optimization techniques for code performance and efficiency
