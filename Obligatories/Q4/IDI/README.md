## Sobre l'Assignatura

Assignatura de 6 crèdits que cobreix dues àrees principals: disseny d'interfícies amb estudi d'usabilitat, i gràfics interactius.

### Temes Principals

**Programació i Gràfics:**
- Desenvolupament de GUI amb Qt (senyals, slots, widgets)
- Processament de geometria 3D i malles de triangles
- Pipeline de visualització OpenGL i transformacions
- Programació de GPU amb shaders GLSL
- Models de color i il·luminació

**Disseny i Usabilitat:**
- Principis de disseny centrat en l'usuari
- Metodologies d'avaluació d'interfícies
- Disseny d'interfícies mòbils
- Mètriques d'usabilitat i revisions d'experts
- Introducció a sistemes AR/VR

### Objectius d'Aprenentatge
- Programar interfícies utilitzant C++ i APIs com Qt
- Dominar conceptes fonamentals de gràfics per computador i implementar visualització 3D en OpenGL
- Comprendre l'arquitectura moderna de GPU i programació GLSL
- Dissenyar aplicacions per a plataformes mòbils/tauletes amb principis d'usabilitat adequats
- Avaluar la usabilitat d'interfícies rigorosament

## About the Subject

This is a 6-credit course covering two main areas: interface design with usability study, and interactive graphics.

### Main Topics

**Programming & Graphics:**
- GUI development with Qt (signals, slots, widgets)
- 3D geometry processing and triangle meshes
- OpenGL visualization pipeline and transformations
- GPU programming with GLSL shaders
- Color models and lighting

**Design & Usability:**
- User-centered design principles
- Interface evaluation methodologies
- Mobile interface design
- Usability metrics and expert reviews
- Introduction to AR/VR systems

### Learning Objectives
- Program interfaces using C++ and APIs like Qt
- Master fundamental computer graphics concepts and implement 3D visualization in OpenGL
- Understand modern GPU architecture and GLSL programming
- Design applications for mobile/tablet platforms with proper usability principles
- Evaluate interface usability rigorously
