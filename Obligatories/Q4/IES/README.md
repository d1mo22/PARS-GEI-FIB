# IES - Introducció a l'Enginyeria del Software

<details open>
<summary>Català</summary>

## Sobre l'Assignatura

Assignatura de 6 crèdits que proporciona una introducció completa a l'enginyeria del software, emfatitzant les etapes d'especificació i disseny. Els estudiants aprenen a especificar sistemes simples utilitzant notació UML i dissenyar-los aplicant patrons arquitectònics i metodologies de disseny.

### Temes Principals
- **Fonaments d'Enginyeria del Software**: Introducció a la disciplina
- **Requisits i Especificació**: Comprensió i documentació de les necessitats del sistema
- **Models de Casos d'Ús**: Modelització del comportament en UML
- **Esquemes Estructurals i de Comportament**: Representació d'estructures de dades i comportament del sistema
- **Disseny de Software**: Principis de disseny orientat a objectes
- **Patrons de Disseny**: Solucions reutilitzables per a problemes de disseny comuns

### Objectius d'Aprenentatge
- Comprendre el procés global d'enginyeria del software
- Comprendre els requisits del sistema i relacionar-los amb les especificacions
- Escriure especificacions de sistemes de software en UML
- Analitzar la completesa i consistència de les especificacions
- Dominar els principis de disseny orientat a objectes utilitzant UML
- Aplicar patrons de disseny comuns
- Comprendre conceptes bàsics de testing de software

## About the Subject

This 6-credit course provides a comprehensive introduction to software engineering, emphasizing specification and design stages. Students learn to specify simple systems using UML notation and design them applying architectural patterns and design methodologies.

### Main Topics
- **Software Engineering Fundamentals**: Introduction to the discipline
- **Requirements & Specification**: Understanding and documenting system needs
- **Use Case Models**: Behavioral modeling in UML
- **Structural & Behavioral Schemas**: Data structure and system behavior representation
- **Software Design**: Object-oriented design principles
- **Design Patterns**: Reusable solutions for common design problems

### Learning Objectives
- Understand the overall software engineering process
- Comprehend system requirements and relate them to specifications
- Write software system specifications in UML
- Analyze specification completeness and consistency
- Master object-oriented design principles using UML
- Apply common design patterns
- Grasp basic software testing concepts
