# IES - Introduccio a l'Enginyeria del Software (Introduction to Software Engineering)

## About the Subject

This 6-credit course provides a comprehensive introduction to software engineering, emphasizing specification and design stages. Students learn to specify simple systems using UML notation and design them applying architectural patterns and design methodologies.

### Main Topics
- **Software Engineering Fundamentals**: Introduction to the discipline
- **Requirements & Specification**: Understanding and documenting system needs
- **Use Case Models**: Behavioral modeling in UML
- **Structural & Behavioral Schemas**: Data structure and system behavior representation
- **Software Design**: Object-oriented design principles
- **Design Patterns**: Reusable solutions for common design problems

### Learning Objectives
- Understand the overall software engineering process
- Comprehend system requirements and relate them to specifications
- Write software system specifications in UML
- Analyze specification completeness and consistency
- Master object-oriented design principles using UML
- Apply common design patterns
- Grasp basic software testing concepts
