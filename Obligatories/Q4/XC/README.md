## Sobre l'Assignatura

Assignatura obligatòria fonamental de 6 crèdits en xarxes de computadors centrada en l'arquitectura i protocols d'Internet.

### Temes Principals
- **Introducció**: Fonaments d'Internet, xarxes de paquets, model de referència OSI
- **Xarxes IP**: Datagrames, adreçament, subxarxes, ICMP, ARP, DHCP, algorismes d'encaminament, NAT, tallafocs
- **Protocol TCP**: Funcions de la capa de transport, recuperació d'errors (ARQ), protocols UDP/TCP, control de flux, control de congestió
- **Xarxes d'Àrea Local**: Mitjans de transmissió, Ethernet, commutadors, VLANs, WiFi (802.11)
- **Aplicacions de Xarxa**: Paradigma client-servidor, HTTP, SMTP, DNS; formats de documents (HTML, XML)

### Objectius d'Aprenentatge
- Identificar funcions dels protocols a través de les capes física, enllaç, xarxa, transport i aplicació
- Comprendre aplicacions client-servidor i adreçament de xarxa
- Predir el comportament dels protocols i interpretar el contingut dels missatges
- Dissenyar configuracions de xarxes IP i configuracions de tallafocs
- Analitzar mecanismes de control de flux i congestió de TCP
- Avaluar el rendiment i topologia de xarxes d'àrea local

## Sobre la Asignatura

Asignatura obligatoria fundamental de 6 créditos en redes de computadores centrada en la arquitectura y protocolos de Internet.

### Temas Principales
- **Introducción**: Fundamentos de Internet, redes de paquetes, modelo de referencia OSI
- **Redes IP**: Datagramas, direccionamiento, subredes, ICMP, ARP, DHCP, algoritmos de encaminamiento, NAT, cortafuegos
- **Protocolo TCP**: Funciones de la capa de transporte, recuperación de errores (ARQ), protocolos UDP/TCP, control de flujo, control de congestión
- **Redes de Área Local**: Medios de transmisión, Ethernet, conmutadores, VLANs, WiFi (802.11)
- **Aplicaciones de Red**: Paradigma cliente-servidor, HTTP, SMTP, DNS; formatos de documentos (HTML, XML)

### Objetivos de Aprendizaje
- Identificar funciones de los protocolos a través de las capas física, enlace, red, transporte y aplicación
- Comprender aplicaciones cliente-servidor y direccionamiento de red
- Predecir el comportamiento de los protocolos e interpretar el contenido de los mensajes
- Diseñar configuraciones de redes IP y configuraciones de cortafuegos
- Analizar mecanismos de control de flujo y congestión de TCP
- Evaluar el rendimiento y topología de redes de área local

## About the Subject

This is a foundational 6-credit mandatory course in computer networking focused on Internet architecture and protocols.

### Main Topics
- **Introduction**: Internet basics, packet networks, OSI reference model
- **IP Networks**: Datagrams, addressing, subnetting, ICMP, ARP, DHCP, routing algorithms, NAT, firewalls
- **TCP Protocol**: Transport layer functions, error recovery (ARQ), UDP/TCP protocols, flow control, congestion control
- **Local Area Networks**: Transmission media, Ethernet, switches, VLANs, WiFi (802.11)
- **Network Applications**: Client-server paradigm, HTTP, SMTP, DNS; document formats (HTML, XML)

### Learning Objectives
- Identify functions of protocols across physical, data link, network, transport, and application layers
- Understand client-server applications and network addressing
- Predict protocol behavior and interpret message content
- Design IP network configurations and firewall settings
- Analyze TCP flow control and congestion mechanisms
- Evaluate local area network performance and topology
