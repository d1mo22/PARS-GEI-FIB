## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits que ensenya conceptes fonamentals de programació paral·lela essencials per a arquitectures de computació multi-nucli modernes.

### Temes Principals
- **Introducció i Motivació**: Necessitat del paral·lelisme, problemes de concurrència (deadlock, curses de dades)
- **Anàlisi de Rendiment**: Mètriques com speedup, escalabilitat; eines de visualització (Paraver, Tareador)
- **Programació de Memòria Compartida**: Implementació OpenMP per a gestió de fils i tasques
- **Arquitectures Paral·leles**: Paral·lelisme a nivell d'instrucció, de dades i de fils; sistemes multiprocessadors
- **Descomposició de Tasques**: Anàlisi de granularitat, identificació de dependències, patrons iteratius vs. dividir i vèncer
- **Descomposició de Dades**: Enfocaments geomètrics i recursius; localitat de dades en sistemes de memòria compartida

### Objectius d'Aprenentatge
- Formular models de rendiment i mesurar l'eficiència d'execució
- Implementar programes paral·lels amb OpenMP
- Identificar estratègies de descomposició apropiades
- Aplicar tècniques de sincronització
- Optimitzar fragments de codi paral·lel

## About the Subject

This 6-credit mandatory course teaches fundamental parallel programming concepts essential for modern multi-core computing architectures.

### Main Topics
- **Introduction & Motivation**: Necessity of parallelism, concurrency issues (deadlock, data races)
- **Performance Analysis**: Metrics like speedup, scalability; visualization tools (Paraver, Tareador)
- **Shared-Memory Programming**: OpenMP implementation for threads and task management
- **Parallel Architectures**: Instruction-level, data-level, and thread-level parallelism; multiprocessor systems
- **Task Decomposition**: Granularity analysis, dependency identification, iterative vs. divide-and-conquer patterns
- **Data Decomposition**: Geometric and recursive approaches; data locality in shared-memory systems

### Learning Objectives
- Formulate performance models and measure execution efficiency
- Implement OpenMP parallel programs
- Identify appropriate decomposition strategies
- Apply synchronization techniques
- Optimize parallel code fragments
