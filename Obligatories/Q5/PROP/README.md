## Sobre l'Assignatura

Assignatura obligatòria de 6 crèdits centrada en l'execució d'un projecte de programació de mitjana escala emfatitzant la fase d'implementació del desenvolupament de software. Els estudiants treballen en equips de 3-4 persones.

### Temes Principals
- Cicle de vida del software i especificacions UML
- Conceptes de programació orientada a objectes en Java
- Disseny i implementació d'arquitectura de tres capes
- Testing de programes, depuració i framework JUnit
- Patrons de disseny (decorator, state, singleton)
- Principis de disseny de GUI
- Eines de control de versions (Git)

### Objectius d'Aprenentatge
- Aplicar coneixements previs de programació per desenvolupar programes correctes i eficients
- Organitzar el treball en equip segons un calendari predeterminat
- Dominar principis de disseny orientat a objectes i implementació en Java
- Gestionar situacions anòmales mitjançant mecanismes d'excepcions
- Crear interfícies d'usuari utilitzant biblioteques GUI

## Projecte del Curs: Cerebro Comercial

Sistema d'optimització de distribució de productes utilitzant diversos algorismes (força bruta, 2-aproximació, hill climbing, cerca local iterada) per a la col·locació òptima en prestatgeries.

**Autors**: Edgar Bosque, Marc Escribano, Sergi Metcalf, David Morais

### Inici Ràpid
```bash
./gradlew run --console plain  # Executar l'aplicació
./gradlew test                  # Executar tests
./gradlew build                 # Compilar el projecte
./gradlew javadoc              # Generar documentació
```

## About the Subject

This is a 6-credit mandatory course focused on executing a medium-scale programming project emphasizing the implementation phase of software development. Students work in teams of 3-4 people.

### Main Topics
- Software lifecycle and UML specifications
- Object-oriented programming concepts in Java
- Three-layer architecture design and implementation
- Program testing, debugging, and JUnit framework
- Design patterns (decorator, state, singleton)
- GUI design principles
- Version control tools (Git)

### Learning Objectives
- Apply prior programming knowledge to develop correctly functioning and efficient programs
- Organize teamwork according to a predetermined schedule
- Master object-oriented design principles and Java implementation
- Handle anomalous situations through exception mechanisms
- Create user interfaces using GUI libraries

## Course Project: Cerebro Comercial
