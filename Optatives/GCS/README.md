# GCS - Gestio de la Ciberseguretat (Cybersecurity Management)

## About the Subject

This is an optional 6-credit course that provides elementary knowledge and resources on cybersecurity oriented toward training IT engineering professionals.

### Main Topics

**Part 1 - Foundational Concepts:**
- Basic cybersecurity concepts and attack types
- Risk analysis and system vulnerabilities
- Incident response procedures
- Digital identity and privacy management

**Part 2 - Emerging Technologies:**
- IoT security and smart cities
- Blockchain technology
- AI applications in cybersecurity
- Quantum computing implications

**Part 3 - Practical Labs:**
- Hands-on vulnerability analysis
- Secure network deployment in virtualized environments
- AI tool applications

### Learning Objectives
- Understand cybersecurity basics, cybercrime, and risk/vulnerability analysis
- Identify problems and solutions in current, emerging, and disruptive technologies
- Collaborate effectively on practical exercises
- Complete guided cybersecurity practices
