## Sobre l'Assignatura

Aquesta assignatura optativa de 6 crèdits ensenya als estudiants com convertir dades en coneixement per a la presa de decisions a través de la fase central del descobriment de coneixement en bases de dades (KDD). Combina estadística, aprenentatge automàtic, tècniques de bases de dades i sistemes de decisió.

### Temes Principals
- Visualització de dades i reducció de dimensionalitat
- Mètodes de clustering (particionament, jeràrquic, expectació-maximització)
- Mètodes predictius (regressió, regressió logística, xarxes neuronals)
- Arbres de decisió (CART)
- Generació de regles d'associació (Apriori, Eclat)
- Mètodes bayesians i anàlisi discriminant
- Preprocessament de dades i protocols de validació
- Mètodes d'ensemble (bagging)

### Objectius d'Aprenentatge
- Identificar els principals tipus de problemes en mineria de dades
- Avaluar la qualitat de les dades i les necessitats de preprocessament
- Seleccionar tècniques estadístiques/d'aprenentatge automàtic apropiades
- Implementar algorismes d'aprenentatge
- Avaluar la qualitat dels resultats
- Presentar troballes professionalment

## About the Subject

This optional 6-credit course teaches students how to convert data into knowledge for decision-making through the central phase of knowledge discovery from databases (KDD). It combines statistics, machine learning, database techniques, and decision systems.

### Main Topics
- Data visualization and dimensionality reduction
- Clustering methods (partitioning, hierarchical, expectation-maximization)
- Predictive methods (regression, logistic regression, neural networks)
- Decision trees (CART)
- Association rule generation (Apriori, Eclat)
- Bayesian methods and discriminant analysis
- Data preprocessing and validation protocols
- Ensemble methods (bagging)

### Learning Objectives
- Identify main problem types in data mining
- Evaluate data quality and preprocessing needs
- Select appropriate statistical/machine learning techniques
- Implement learning algorithms
- Assess result quality
- Present findings professionally
