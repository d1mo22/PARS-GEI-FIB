## Sobre les Optatives

Aquesta carpeta conté les assignatures optatives del grau de la FIB.

## Assignatures

| Codi | Assignatura | Descripció |
|------|-------------|------------|
| GCS | [Gestió de la Ciberseguretat](./GCS/) | Gestió de ciberseguretat, seguretat IoT, blockchain |
| MD | [Mineria de Dades](./MD/) | Mineria de dades, clustering, classificació, programació R |
| PAE | [Projecte Aplicat d'Enginyeria](./PAE/) | Projecte d'enginyeria aplicada amb empreses |
| SID | [Sistemes Intel·ligents Distribuïts](./SID/) | Sistemes multiagent, teoria de jocs, aprenentatge per reforç |
| VC | [Visió per Computador](./VC/) | Visió per computador, processament d'imatges, detecció de característiques |

## Sobre les Optatives

Les assignatures optatives permeten als estudiants:
- Explorar àrees d'interès més enllà de la seva especialització
- Guanyar experiència pràctica en projectes (PAE)
- Desenvolupar habilitats interdisciplinàries
- Preparar-se per a carreres professionals específiques

## About Electives

This folder contains optional/elective subjects from the FIB degree.

## Subjects

| Code | Subject | Description |
|------|---------|-------------|
| GCS | [Cybersecurity Management](./GCS/) | Cybersecurity management, IoT security, blockchain |
| MD | [Data Mining](./MD/) | Data mining, clustering, classification, R programming |

| VC | [Computer Vision](./VC/) | Computer vision, image processing, feature detection |

## About Electives

Elective subjects allow students to:
- Explore areas of interest beyond their specialization
- Gain practical project experience (PAE)
- Develop cross-disciplinary skills
- Prepare for specific career paths
