## Sobre l'Assignatura

Aquesta és una assignatura optativa de 6 crèdits enfocada a l'extracció i anàlisi d'informació d'imatges i seqüències d'imatges. Les aplicacions van des del reconeixement facial fins al diagnòstic mèdic, detecció d'objectes, guiatge robòtic i conducció autònoma.

### Temes Principals
- **Fonaments**: Propietats d'imatges digitals, espais de color, discretització
- **Processament**: Filtratge, detecció de vores, operadors morfològics
- **Segmentació**: Tècniques basades en regions i en contorns
- **Descriptors**: Caracterització de formes i característiques amb propietats d'invariància
- **Reconeixement**: Mètodes de classificació (Bayes, k-veïns més propers, PCA)
- **Característiques Locals**: SIFT, detectors Harris, característiques Haar, algorisme Viola-Jones
- **Anàlisi de Moviment**: Flux òptic, seguiment d'objectes, detecció de moviment

### Objectius d'Aprenentatge
- Analitzar, dissenyar, programar i avaluar mètodes d'anàlisi d'imatges
- Treballar amb restriccions de temps de resposta, fiabilitat i requisits de cost
- Aplicar tècniques de visió per computador a problemes reals
- Implementar algorismes de detecció de característiques i classificació

- Trabajar con restricciones de tiempo de respuesta, fiabilidad y requisitos de coste
- Aplicar técnicas de visión por computador a problemas reales

## About the Subject

This is a 6-credit optional course focusing on extracting and analyzing information from images and image sequences. Applications span from face recognition to medical diagnostics, object detection, robotic guidance, and autonomous driving.

### Main Topics
- **Fundamentals**: Digital image properties, color spaces, discretization
- **Processing**: Filtering, edge detection, morphological operators
- **Segmentation**: Region-based and contour-based techniques
- **Descriptors**: Shape and feature characterization with invariance properties
- **Recognition**: Classification methods (Bayes, k-nearest neighbor, PCA)
- **Local Features**: SIFT, Harris detectors, Haar features, Viola-Jones algorithm
- **Motion Analysis**: Optical flow, object tracking, motion detection

### Learning Objectives
- Analyze, design, program and evaluate image analysis methods
- Work with temporal response constraints, reliability, and cost requirements
- Apply computer vision techniques to real-world problems
- Implement feature detection and classification algorithms
